# Figma Landing Page - ShipX

This is a responsive landing page project built with **Laravel** based on a Figma design. The project includes a contact form, dashboard, and REST APIs for CRUD operations.

## Features

- Fully responsive landing page based on Figma design.
- Contact form to submit leads.
- Dashboard to view, edit, and delete submitted leads.
- REST APIs for CRUD operations on leads.
- Laravel validation for form submissions.
- TailwindCSS for UI styling.
